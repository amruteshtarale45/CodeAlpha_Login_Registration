# Login & Registration System

This repository contains a C++ console-based program that allows users to register and log in.

## Features

- User registration: Stores user credentials in a text file (`users.txt`).
- User login: Validates user credentials against stored records.
- Simple console interface for ease of use.

## Language

- C++

## How to Run

1. Compile the program using a C++ compiler:
   ```
   g++ loginReg.cpp -o loginReg
   ```
2. Run the executable and follow the prompts to register a new user or log in with existing credentials.
